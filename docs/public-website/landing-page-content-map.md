# Landing Page Content Map — Aish POS Lite

Sprint 21 — Public Website / Landing Page Readiness Foundation.

Governed content plan for the public landing page. All copy is governance metadata
and must pass content approval before publish. **No unsupported claims.**

## Hero message

- **Headline:** "Kasir Android ringan untuk UMKM Anda."
- **Subheadline:** QRIS, tunai, mode offline, cetak struk, stok sederhana, laporan harian.
- **Primary CTA:** "Ajukan Minat" → `#interest` (interest-only; never account creation).
- **Secondary CTA:** "Lihat Paket" → `/packages`.

## Target segments

Warung, Toko Kecil, Kedai, Laundry, Retail, Apotek ringan, UMKM umum. These mirror
the commercial package catalog segments (Sprint 20).

## Main benefits

1. Jualan lebih cepat di perangkat Android ringan.
2. Terima QRIS dan tunai; tetap jalan saat offline.
3. Struk termal, stok otomatis, laporan harian.

## Feature sections

- Pembayaran QRIS (diproses aman di backend).
- Tunai & mode offline dengan sinkronisasi otomatis.
- Cetak struk ESC/POS.
- Stok sederhana berbasis pergerakan.
- Laporan & tutup kasir harian.
- Multi-perangkat sesuai batas paket.

## Package highlight section

Ringkasan paket diambil dari commercial package catalog (Sprint 20). Harga bersifat
indikatif/tata kelola. **Tidak ada penagihan otomatis.** See
[package-pricing-content-alignment.md](package-pricing-content-alignment.md).

## CTA section

CTA hanya boleh menuju formulir minat (`#interest`), `/packages`, atau `/thank-you`.
CTA **tidak boleh** menuju pembuatan akun, signup mandiri, atau checkout billing.

## FAQ section

- Apakah bisa daftar sendiri? Belum — aktivasi melalui tim kami.
- Apakah ada penagihan otomatis? Tidak pada tahap ini.
- Apakah bisa offline? Ya, transaksi tunai tetap jalan lalu tersinkron.

## Trust / proof placeholder

Placeholder untuk testimoni/logo mitra. Kosong sampai bukti nyata tersedia —
**dilarang** membuat proof point yang tidak dapat diverifikasi.

## No unsupported claims rule

Setiap klaim fitur harus sesuai dengan foundation produk. Klaim performa, keamanan,
atau kepatuhan yang belum terverifikasi dilarang dan dicatat sebagai risiko
`CONTENT_CLAIM` di [public-website-risk-register.md](public-website-risk-register.md).
